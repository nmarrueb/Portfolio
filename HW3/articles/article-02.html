If You're Bored Button Proposal

"If You're Bored Button" will be an application to randomly present a javascript based game everytime you click on the button. This is a similar idea to a popular already existing application called boredbutton. The difference will be that the games will all be hosted under our application and does not link external sites. Some of the games that we use will be made by us. We may plan on implementing other games as well, also a gallery page to select specific games at will. The front end of our development will include an aesthetic and functional UI, the backend and database development will entail hosting, storing player data, and providing the javascript games. Our inspiration for creating our website is to provide a fun time waster website to show off small javascript games in a presentable and interesting way.
